## [v1.5.0] - 仓库内容实时浏览器 (2024-05-20)

### 设计目标
当用户成功连接 GitHub 仓库后，在主界面提供可视化文件列表，让用户能够直观查看仓库内容。

### 核心更新
1.  **双视图模式 (Dual-View Mode)**:
    *   主界面引入了“对话 (Chat)”与“浏览器 (Explorer)”两个选项卡。
    *   用户可以在与助手交流的同时，随时切换查看当前的工程文件结构。
2.  **仓库内容浏览器 (Repo Explorer)**:
    *   集成 GitHub Contents API，动态拉取当前仓库和分支的文件列表。
    *   **图标化识别**: 为文件夹和不同类型的常用文件（如 `.md`, `.json`, `.ts`）提供不同的 Lucide 图标。
    *   **元数据展示**: 显示文件大小及文件类型。
3.  **交互增强**:
    *   配置变更（如切换分支）时，浏览器内容会自动刷新。
    *   空状态下提供友好的连接引导。

### 技术说明
*   新增 `fetchRepositoryContents` 服务函数。
*   使用 Tailwind 实现平滑的视图切换动画。
